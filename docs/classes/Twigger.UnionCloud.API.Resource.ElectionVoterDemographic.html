Enable advanced casting for multiple return variables

customCasts : array
inherited

The key of the array should contain the new attribute reference in snake case, and a string representing the class. i.e. usergroup|UserGroup::class.

The value of the array should be an array to specify the transformation from the first attribute to the second attribute. I.e. if the initial request contains a key called ug_id, but the UserGroup class has an attribute id, we put ['ug_id' => 'id']
